# voicer

一个语音交互的App

1. 提供手机端(iphone，安卓)、web端、iwatch端
2. 可以支持
   1. 搜索本地资源，包含：音频、文本
   2. 接入LLM，可以自定义LLM配置
   3. 可以进行闹钟设置
3. 交互以对话框为主
   1. 提供文本输入
   2. 提供语音输入
   3. 提供电话聊天
4. 对话框设计要求
   1. 提供自定义形象
      1. 头像
      2. 音色
      3. 性格
      4. 兴趣偏好
   2. 对话框设计简洁为主
5. App配置提供两种途径
   1. 形象展示框右上角提供按钮
   2. 语音输入后跳转
